# BMI Calculator App

A Flutter app to calculate Body Mass Index (BMI) and provide a categorical result.

## Features

*   Input height and weight.
*   Calculate BMI.
*   Display BMI result.
*   Categorize BMI (Underweight, Healthy, Overweight, Obese).

## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to start the app.
